# Stochy

Stochy is a desktop application built with JavaFX for efficient data management using PostgreSQL/MySQL.

## 🚀 Features
- User-friendly interface (JavaFX)
- Database integration
- Modular architecture (MVC)

## 🛠 Tech Stack
- Java / JavaFX
- PostgreSQL / MySQL
- Maven (optional)

## 📦 Setup
1. Clone the repo
2. Configure database
3. Run the application
